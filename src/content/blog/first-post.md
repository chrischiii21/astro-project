# Welcome to Astro!

Astro is a modern web framework that helps you build **fast, content-focused websites**.

## Why Choose Astro?

- **Zero JS by default**: Ships only the JavaScript you need
- **Component Islands**: Mix and match UI frameworks
- **Fast builds**: Optimized for speed and performance

## Getting Started

```bash
npm create astro@latest
```

This is just the beginning of your Astro journey!
